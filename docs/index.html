merge

  • merge<T>(...iterables: Array<Iterable<T> | AsyncIterable<T>>): AsyncIterableIterator<T>
    • Defined in index.ts:185

    Type parameters

    • T

    Parameters

    • Rest ...iterables: Array<Iterable<T> | AsyncIterable<T>>

    Returns AsyncIterableIterator<T>

    an iterable that yields values from all provided iterables as they become available.

    When the iterables provided are synchronous, merge is equivalent to interleave, but when one or more of the iterables are asynchronous, values will be yielded from each iterable as soon as they are ready.

    Inspired by RxJS's merge operator.
